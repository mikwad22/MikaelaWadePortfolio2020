The title: Ur-A-Critic

A Movie Application

This is a Red Badge Project that was built during the last few weeks of Web Development Bootcamp at Eleven Fifty Academy. As a team we created a full CRUD app using Legacy React, TypeScript, and JavaScript fundamentals that allows users to signup, login and browse a list of featured movies and a list of weekly recommendations added by the Admin, with the ability to add to favorites, delete from favorites, add a comment, edit a comment, and delete a comment.

In the admin view, the Admin is able to login and add movies to the recommended weekly movie list (viewed by the user), delete a movie from the list, see a list of active members(users), update a members name, and delete a member. This project utilizes The Movie Database to fetch movies from their API.
